# Portfolio Organization

This organization contains both my previous and current portfolio projects, highlighting my growth as a full stack developer and the evolution of my tech stack over time.

## Overview

Inside this organization, you will find two portfolio versions:

- **Old Portfolio** — built with **HTML, CSS, and JavaScript**
- **New Portfolio** — built with **TypeScript, React, Tailwind CSS, Tailwind UI, and shadcn/ui**

The goal of keeping both projects here is to document my progress, preserve earlier work, and showcase how my development approach has changed with experience.

## Old Portfolio

The old portfolio was built using fundamental web technologies:

- HTML
- CSS
- JavaScript

This project reflects my early approach to web development, with a focus on building layouts, styling pages, and adding interactivity using vanilla JavaScript.

## New Portfolio

The new portfolio is a redesigned and modernized version built with:

- TypeScript
- React
- Tailwind CSS
- Tailwind UI
- shadcn/ui

This version improves on the previous one by using a more scalable and maintainable architecture, reusable UI components, stronger type safety, and a more modern design system.

## Why This Organization Exists

This organization was created to:

- Keep both portfolio versions in one place
- Show the transition from traditional frontend development to a modern stack
- Demonstrate personal growth and learning
- Preserve the history of my work

## Growth as a Developer

Comparing these two projects shows my journey from building websites with core web technologies to creating applications with modern tools and frameworks. Both portfolios are valuable, as they represent different stages of my learning and development.
